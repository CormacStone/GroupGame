# Game Ideas
* Quest is a cookbook with silhouettes that get filled in
* Animated sprites for Rat and Mini-Bosses
* Mousetraps

# Levels
* Grandma's House (tutorial)
* Pantry
* Sewers (Home)
* Stovetops
* Dishwashing
* Dining Area
* Head Chef's Office

# Missions
* Find recipe ingredients (Main)
* Find lost rats (Side)
* Sewer runs (Arcade)
* Help dog (Side)
